Jawaban No 1 UAs

Web Server: Sebuah komputer yang memiliki perangkat lunak khusus (seperti Apache, Nginx, Microsoft IIS) yang bertanggung jawab untuk menyimpan, mengelola, dan menyajikan konten dari sebuah website kepada pengguna internet. Web server menerima permintaan dari browser pengguna, kemudian mengirimkan file atau halaman web yang diminta tersebut melalui internet. Domain: Merupakan alamat unik yang digunakan untuk mengidentifikasi sebuah website di internet. Misalnya, "www.nama-website.com". Domain memungkinkan pengguna internet untuk mengakses website dengan mudah tanpa harus mengingat alamat IP yang panjang. Hosting: Layanan penyimpanan dan pemuatan (hosting) file, data, dan konten sebuah website ke dalam server yang terhubung ke internet. Hosting menyediakan tempat bagi website untuk berada online sehingga dapat diakses oleh pengguna internet dari berbagai lokasi. Hosting dapat dianggap sebagai "rumah" bagi website di internet, tempat di mana file-file website disimpan dan diakses oleh pengguna. Hubungan Antara Ketiganya: Ketika seseorang mengakses suatu website, browser akan mengirimkan permintaan ke domain yang terhubung ke server tempat website tersebut di-hosting. Domain akan meneruskan permintaan ke alamat IP yang terkait dengan server tempat website tersebut di-hosting. Web server di server hosting akan merespons permintaan dari browser dengan mengirimkan konten website yang diminta, seperti halaman web, gambar, video, atau file lainnya kembali ke browser pengguna.
